# Artifacts

This directory holds tracked non-runtime assets that are useful for development but should not live in the repo root.

- `validation/`
  Browser screenshots and visual regression snapshots.
- `sprite-debug/`
  Sprite extraction experiments, crop candidates, zoomed regions, and cleanup references.
- `debug/`
  DOM captures, fleet JSON dumps, and other troubleshooting output.

Runtime PixelOffice assets stay under `packages/web/public/pixel-office`.
